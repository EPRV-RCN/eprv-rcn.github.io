EPRV ADS Libraries

We are in the process of assembling a series of ADS Libraries that capture EPRV-related papers across a variety of topics (instrumentation, data reduction pipelines, stellar activity mitigation, planet discoveries, etc) and need suggestions from the community! Please use this ADS Libraries Google form to provide suggested papers and we will update this page once the first draft of the libraries are set up and ready for use.

Interested in improving this page? Create a fork and make changes to this source file, then submit a pull request!
